Bienvenido a la Calculadora

Descripción del Proyecto

Esta aplicación de calculadora fue desarrollada en Java y permite realizar operaciones matemáticas básicas. La aplicación se despliega en un entorno Docker con una configuración de múltiples etapas y un proxy inverso. A continuación, puedes encontrar enlaces a la documentación, resultados de pruebas y cobertura de código.
